## 🚀 DEPLOYMENT COMMANDS

### **IMMEDIATE PRODUCTION DEPLOYMENT:**

```bash
# 1. Start Backend
cd backend
go run main.go

# 2. Start Database
docker run -d -p 5432:5432 -e POSTGRES_DB=agroai postgres:13

# 3. Run Migrations
psql -h localhost -U postgres -d agroai -f schema.sql

# 4. Seed Data
psql -h localhost -U postgres -d agroai -f db/seeds/seed_all_data.sql

# 5. Start Frontend
cd client
npm start
```

### **PRODUCTION ENVIRONMENT:**
```bash
# Docker Deployment
docker-compose -f docker-compose.prod.yml up -d

# Health Check
curl http://localhost:8080/api/health
```
